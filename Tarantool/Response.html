    struct Tarantool::Response

    • Tarantool::Response
    • Struct
    • Value
    • Object

    Overview

    A response sent by Tarantool. It should not be initialized manually.

    See Tarantool docs to know more about responses.

    Example:

    response = db.select(999, 0, {1})
if response.success?
  puts response.body.data
end
# => [[1, "hello world"]]
